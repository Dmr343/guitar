# Atajos de teclado

## CAGED — 5 Formas (`acordes.html`)

| Tecla | Acción |
|-------|--------|
| `1` / `2` / `3` / `4` | Cambiar de modo (Forma / Mapa / Zona / Progresión) |
| `C` `A` `G` `E` `D` | Modos 1, 3, 4: seleccionar forma · Modo 2: togglear forma |
| `←` / `→` | Modo 1: mover forma un semitono · Modo 4: acorde anterior / siguiente |
| `Espacio` | Modo 4: play / pause metrónomo |
| `M` | Modo 4: toggle metrónomo auto / manual |
| `Shift+↑` / `Shift+↓` | BPM +5 / −5 (afecta el metro del Modo 4) |
| `Escape` | Detener metrónomo |

## Improvisar (`improvisar.html`)

| Tecla | Acción |
|-------|--------|
| `3` / Numpad `3` | Toggle Tríadas 3 cuerdas |
| `2` / Numpad `2` | Toggle Tríadas 2 cuerdas |
| `7` / Numpad `7` | Toggle 7mas diatónicas |
| `S` | Toggle Sus2 (desactiva Sus4) |
| `D` | Toggle Sus4 (desactiva Sus2) |
| `←` | Posición anterior |
| `→` | Posición siguiente |
| `Escape` | Resetear filtro de posición a "Todas" |
